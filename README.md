## Get started

* `launch "Unrailed!"`
* `set the resolution to 800x600 px`
* `set the first player to the yellow Penguin with the azerty layout`
* `git clone https://github.com/Naexys/Unrailed_AI.git`
* `cd Unrailed_AI/src`
* `python main.py`
